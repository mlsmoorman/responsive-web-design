HTML & CSS

HTML or HyperText Markup Language defines the webpage content.

CSS or Cascading Style Sheets is used to style the content of the webpage.

HTML Terms
  • Elements

    Designators that define the structure and content of the objects inside of a web-page. Here is an example of an anchor element: < a >

  • Tags

    Elements are wrapped in less than and greater than brackets - these are known as Tags. There are opening and closing tags. Here is an example of an anchor link with opening and closing tags: < a > ... < / a >

  • Attributes

    Attributes provide additional information about an element. They are also used to identify content that you want to change or manipulate using CSS. Sticking with our same example, the anchor tag, here you will ssee an attribute of "href" which is used to define the new location the anchor tag will direct you to: < a href="https://www.google.com/" < / a >

CSS Terms
  • Selectors

    A selector determines which element(s) of a page to target and apply CSS styles to. For example, if you wanted to apply CSS styling to all of your paragraph elements in your app, it would look like this: p { ... }

  • Properties

    Once an element is targeted you can update the properties within the selector to change its properties. In the below example we are using the background-color property to change the background color to black: p { background-color: black; }

  • Values

    Once we have targeted the element to be modified and have determined which property we would like to adjust, we add a value. In the above example, the value we used would be "black."

HTML Structure

HTML documents are saved as .html files. There are several text editors that can be used available for free. I defer to using VS Code. You can choose from the many that are out there.

The structure of the file must start with < !DOCTYPE html>

along with an html element that houses head and body elements.

The head is where your meta information goes along with your document title and any links to your CSS stylesheets.

The body is where the bulk of your HTML code will be housed - this is where all of the information displayed on your webpage shows up.

CSS Selectors
  • Class Selectors

    The class selector targets a group of elements and not just all of the elements of the same type. The class selector allows changing properties of different parts of the HTML document regardless of the element type, as long as the class selector is the same for those targeted elements.

    CSS example:

    .purple { ... }

    HTML example:

    <div class="purple"> <p class="purple">

  • ID Selectors

    ID selectors are more precise than class selectors. ID elements should only be used once per page and only within similar elements.

    CSS example:

    #purple { ... }

    HTML example:

    <div id="purple">
